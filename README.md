# Hôtel Célébrités — Site

Site vitrine de l'**Hôtel Célébrités** — hôtel, restaurant, bar et salles de réception à Agbanou, Ouidah, Bénin. Inauguré en 2022.

## Aperçu local

Double-clic sur `index.html` pour ouvrir dans le navigateur.

## Déploiement

Site statique — aucun build, aucune dépendance serveur.
Compatible Netlify, Vercel, Cloudflare Pages, GitHub Pages.
